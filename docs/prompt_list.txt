google sheets api를 사용해서 스프레드 시트에 있는 시트의 내용을 출력하고 싶어 backend에서 service_account.json을 참조해서 FastAPI로 시트 정보를 가져오고 frontend에 거져온 시트 를 출력해줘 그리고 스프레드 시트 ID 는 1gVBHcmkSASco3rDXxjTGXfaMjiWRjYtavBDVP6o41ss 이거야
이제 여기서 보여지는 시트의 행과 열을 삭제하거나 추가하능 기능을 만들어줘
열 추가를 할때는 제목을 입력받아서 해당 제목으로 열을 추가했으면 좋겠어 그리고 행과 열을 추가할때 .env에 있는 openapikey값을 활용해서 시트의 정보를 주고 안에 들어갈 적절한 내용을 넣어서 추가해줘
이제 AI자동 시트 생성기능을 만들고 싶어 시트를 생성하는 버튼을 하나 만들고 버튼을 누르면 프롬프트 채팅창을 띄우고 거기에다가 어떤 시트를 만들지 입력하면 해당 시트를 만들어 주는 기능을 추가해줘
지금 main.py에 너무 스크립트가 몰리고 있어 기능별로 폴더만들어서 스크립트 나눠서 정리해줘 그리고 되돌리기 버튼을 만들고 버튼을 누르면 실행 전으로 되돌리는 기능도 구현해줘
AI수정버튼을 만들어서 프롬프트로 테이블 안의 셀 내용을 수정할수 있게 만들어줘
.env에 있는 DB정보를 이용해서 hitory테이블을 생성해서 수정사항을 히스토리에 저장해서 보고 다시 돌릴수 있게 구현해줘
이제 선택된 해당시트를 다운로드 하는 기능도 추가했으면 좋겠어 다운로드는 json파일과 csv파일 두개 만들어줘
